# 🤖 Smart AI Resume Builder & Analyzer

<p align="center">
  <img src="images/banner.png" alt="Smart AI Resume Builder & Analyzer" width="100%">
</p>

<p align="center">
An AI-powered Resume Builder and ATS Resume Analyzer that helps users create professional resumes, evaluate ATS compatibility, identify missing skills, and receive AI-generated resume improvement suggestions.
</p>

---

## ✨ Features

* 📄 Build professional resumes
* 📥 Download resumes as PDF and DOCX
* 📊 ATS Resume Score (out of 100)
* 🤖 AI-powered Resume Analysis using Ollama
* 💡 Resume strengths and weaknesses
* 📈 Missing skills detection
* 🎯 Resume improvement suggestions
* 👤 User Login & Signup
* 🔐 Admin Dashboard
* 💾 User Database Management

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/naveenmalali/Smart-AI-Resume-Builder-Analyzer.git
```

```bash
cd Smart-AI-Resume-Builder-Analyzer
```

### Install frontend dependencies

```bash
npm install
```

### Install backend dependencies

```bash
cd ai-proxy-server
npm install
```

---

## ▶️ Run the Project

### Start the backend

```bash
cd ai-proxy-server
node index.js
```

You should see:

```text
✅ Server running on http://localhost:5000
```

### Start the frontend

Open another terminal:

```bash
npm start
```

The application will open at:

```text
http://localhost:3000
```
